Thanks for downloading the Billboard Mage 1.0 extension for Unity.

Recommended Unity version 2021.3.14f1 or higher

Credit: https://sharpcoderblog.com/

You are permitted to use this extension in free or commercial projects.

How to use:

"Billboard Mage 1.0 -> Settings" to tweak the generated billboard settings

To generate a billboard select a 3D object in Scene view then go to "Billboard Mage 1.0 -> Create Billboard" Asset then select the path where to store BillboardAsset, Material, and Billboard Texture, after that, a new object will be created in the exact same spot with the BillboardRenderer component attached.
